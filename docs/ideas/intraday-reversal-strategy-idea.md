# 아이디어: 급락 후 급반등 포착 전략

## 발단
Missed Entries 화면에서 S3/S4 탈락 종목 중 당일 장중고점 +12%, 장중저점 -13%를 모두 찍은 종목들이 개선후보로 잡힘.
예: 일정실업(008500) 기준가 827원, 장중저점 -13.54%, 장중고점 +12.45%

→ 저점에서 반등을 포착해서 매수할 수 있지 않을까?

## 개념

- 기존 전략(Momentum): 상승 중인 종목에 올라타기
- 이 아이디어(Reversal): 급락 중인 종목이 바닥에서 반등하는 시점에 진입

## 인프라 가용성

- WS 실시간 틱 → 가격 반전 시점 감지 가능
- shnu_rate(체결강도) → 매수세 전환 포착 가능
- 10초봉 집계 → 단기 추세 반전 패턴 가능
- S6 decision_engine에 신규 조건 그룹으로 추가 가능

## 핵심 과제

- 바닥 판단이 어려움 — -13%가 바닥인지 -20%까지 더 빠지는지 알 수 없음
- 페이크 반등 필터링 — 체결강도 반전만으로는 오신호 많음
- 손절 로직 별도 설계 필요 — 반등 실패 시 즉시 손절 (트레일링 스탑과 다름)
- Universe 확장 필요 — 현재 S3 탈락 종목은 진입 대상 밖

## 상태

아이디어 수준. 기획/구현 미착수.
