🌐 1단계: Cursor에서 Ubuntu 서버로 SSH 원격 접속하기
서버에 직접 모니터와 키보드를 연결하는 대신, 내 PC의 Cursor 화면을 통해 서버의 내부로 원격 접속(SSH)하는 과정입니다.
Cursor IDE 실행 및 확장 프로그램 설치
Cursor 프로그램의 왼쪽 메뉴 바에서 **'Extensions(확장 프로그램, 퍼즐 모양 아이콘)'**을 클릭합니다.
검색창에 Remote - SSH를 검색하고 설치합니다. (Microsoft에서 만든 공식 플러그인입니다)
서버에 연결하기
화면 왼쪽 아래 구석을 보면 >< 모양의 초록색(또는 파란색) 아이콘이 있습니다. 이를 클릭합니다.
화면 위쪽 중앙에 검색창 같은 것이 열리면, **Connect to Host... (호스트에 연결...)**를 선택합니다.
이어서 **Add New SSH Host... (새 SSH 호스트 추가...)**를 클릭합니다.
연결 명령어를 입력하라고 나오면, 대여하신 서버의 접속 정보를 바탕으로 아래 형식처럼 입력하고 Enter를 누릅니다.
ssh 계정명@서버의IP주소 (예: ssh root@123.45.67.89 또는 ssh ubuntu@123.45.67.89)
비밀번호 입력 및 접속 확인
서버의 운영체제를 묻는 창이 뜨면 **Linux**를 선택합니다.
이어서 서버의 비밀번호를 입력하고 Enter를 누릅니다.
왼쪽 아래 >< 아이콘 옆에 SSH: 서버IP 라고 표시되면 성공적으로 원격 접속된 것입니다!
상단 메뉴의 File -> Open Folder를 눌러서 작업할 기본 폴더(예: /home/ubuntu/ 또는 /root/)를 열어주세요.
--------------------------------------------------------------------------------
🛠️ 2단계: Ubuntu 서버 환경 세팅 및 한투 AI 플러그인 설치
이제 서버 내부에 파이썬과 한국투자증권이 공식 지원하는 AI 코딩 에이전트용 확장 플러그인을 설치하겠습니다. Cursor 하단에 터미널을 열고(단축키: Ctrl + `) 아래 명령어들을 한 줄씩 복사해서 붙여넣고 Enter를 쳐주세요.
1. 서버 시스템 업데이트 및 필수 프로그램(Node.js 등) 설치 플러그인 설치를 위해 Node.js 환경이 필요합니다
.
sudo apt update && sudo apt upgrade -y
sudo apt install -y python3-pip python3-venv nodejs npm curl
2. 초고속 파이썬 패키지 관리자 uv 설치 한국투자증권 공식 샘플 환경에서는 빠르고 간편한 의존성 관리를 위해 uv 패키지 매니저 사용을 권장하고 있습니다
. 터미널에 아래 명령어를 입력하여 설치합니다
.
curl -LsSf https://astral.sh/uv/install.sh | sh
(설치 후 터미널을 껐다가 다시 열거나 source $HOME/.cargo/env를 입력해 적용해주세요)
3. KIS(한국투자증권) Quant Plugin - Cursor 전용 에이전트 설치 한국투자증권 오픈 API를 Cursor AI가 직접 이해하고 자연어로 전략 설계부터 백테스트, 주문까지 도와주도록 설정하는 플러그인을 설치합니다
. 터미널에 다음 명령어를 입력하세요.
npx @koreainvestment/kis-quant-plugin init --agent cursor
이 명령어를 실행하면 Cursor 전용 훅(hook)과 규칙 파일(.cursor/rules/kis-safety.mdc)이 서버 폴더에 자동으로 생성되며, 봇이 민감 정보(App Key 등)를 유출하지 않도록 차단하고 실전 주문 시 반드시 사용자 승인을 거치도록 안전장치가 세팅됩니다
.
