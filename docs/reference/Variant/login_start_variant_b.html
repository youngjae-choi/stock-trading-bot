이 화면은 사용자 인증 이후, 계좌와 운영 모드를 확인하고 홈 대시보드로 들어가기 전 마지막 확인을 수행하는 시작 화면입니다.

1. OAuth 로그인 사용자 세션을 만들고 서비스 접근 권한을 확인합니다.
2. 계좌 및 모드 확인 모의/실전 모드와 기본 연결 상태를 확인합니다.
3. 홈으로 진입 전략, 운용 상태, 시스템 연결 화면으로 이동합니다.
