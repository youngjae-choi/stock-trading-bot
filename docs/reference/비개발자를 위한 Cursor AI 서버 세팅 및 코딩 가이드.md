비개발자 입장에서 리눅스 서버를 세팅하고 코딩을 진행하는 것은 막막할 수 있습니다. 하지만 Cursor IDE와 같은 AI 코딩 어시스턴트를 활용하면 질문만 잘해도 완벽한 코드를 얻어낼 수 있습니다.
가장 먼저 바로잡아야 할 중요한 개념 하나와, AI에게 지시하는 완벽한 프롬프트(명령어) 작성법, 그리고 비개발자도 할 수 있는 텍스트 기반 화면 기획서 작성법을 정리해 드립니다.
1. 💡 개념 바로잡기: 리눅스 서버에 IDE를 직접 설치하지 않습니다!
개발자들은 리눅스(클라우드 서버) 화면에 직접 들어가서 코딩하지 않습니다.
올바른 세팅 방법: 사용자님의 윈도우/맥 PC에 Cursor IDE를 설치한 뒤, Cursor의 'Remote-SSH(원격 접속)' 기능을 이용해 리눅스 서버에 선을 연결하듯 원격으로 접속하여 코딩합니다.
이렇게 하면 내 PC의 편안한 화면에서 작업하지만, 실제 파일이 저장되고 코드가 실행되는 곳은 24시간 돌아가는 리눅스 서버가 됩니다.
--------------------------------------------------------------------------------
2. 🤖 AI(Cursor)에게 서버 세팅 및 코딩 지시하는 방법 (프롬프트 공식)
비개발자가 AI에게 지시할 때는 **"역할 부여 + 단계별 지시 + 기술 스택 명시"**가 핵심입니다. Cursor의 채팅창(Ctrl+L)에 아래와 같이 복사해서 붙여넣어 보세요.
[프롬프트(명령어) 템플릿]
"너는 파이썬(Python) 기반 퀀트 트레이딩 시스템을 구축하는 20년 차 시니어 개발자야. 나는 코딩을 모르는 비개발자니까 명령어와 터미널 입력값을 아주 쉽고 정확하게 알려줘야 해.
[목표] 우분투(Ubuntu) 리눅스 서버에 한국투자증권 오픈 API와 Streamlit을 활용한 자동매매 봇을 세팅할 거야.
[현재 상황] 내 로컬 PC에 Cursor를 설치했고, 리눅스 서버 대여까지 마쳤어.
[지시 사항] 한 번에 모든 코드를 짜지 말고, 아래 순서대로 하나씩 대화를 나누며 진행해 줘. 내가 '다음'이라고 하면 다음 단계로 넘어가.
내 PC의 Cursor에서 리눅스 서버로 SSH 원격 접속하는 방법 알려주기
리눅스 서버에 파이썬 3.11 환경 세팅 및 한투 API AI 에이전트 확장 플러그인(npx @koreainvestment/kis-quant-plugin init --agent cursor) 설치하기
프로젝트 폴더 구조 설계하기
한국투자증권 API 초당 호출 제한(EGW00201) 방어를 위한 안전한 호출 함수 만들기
Streamlit 기반 UI 만들기"
--------------------------------------------------------------------------------
3. 🖥️ 비개발자를 위한 '텍스트 기반 화면 기획서' 만드는 법
Figma나 PPT 같은 복잡한 툴을 쓸 필요가 없습니다. Streamlit은 파이썬 코드로 레이아웃을 짜기 때문에, AI에게 컨테이너(구역)와 컬럼(단)을 어떻게 나눌지 글(텍스트)로 설명하는 것이 가장 정확합니다.
유명 트레이딩 봇 UI 구현 사례를 참고하여
, 아래와 같은 **'텍스트 기획서'**를 Cursor AI에게 던져주면 알아서 반응형 웹 코드를 짜줍니다.
[Streamlit 대시보드 텍스트 기획서 템플릿]
"UI 대시보드는 Streamlit을 사용해서 PC와 모바일 모두 보기 좋은 반응형(layout="wide")으로 만들어줘
. 아래의 기획서 구조에 맞춰서 st.container와 st.columns를 활용해 레이아웃을 그려줘
.
[화면 상단: 헤더 컨테이너]
제목: "📈 하이브리드 트레이딩 & 리밸런싱 센터"
[화면 중단 1: 봇 제어 컨테이너 (4개 컬럼으로 쪼개기)]
컬럼 1: 거래 플랫폼 선택 드롭다운 (한국투자증권, 모의투자 등)
컬럼 2: 알림 수신 채널 선택 (Telegram)
컬럼 3: API 토큰 발급 상태 및 갱신 버튼
컬럼 4: [단타 봇 자동매매 시작/정지] 스위치 토글 (기본값: False)
[화면 중단 2: 실시간 모니터링 컨테이너 (2개 컬럼)]
컬럼 1: 현재 계좌 잔고 및 총 평가 금액 (크게 표시)
컬럼 2: 오늘 단타 봇이 벌어들인 누적 수익금 및 체결 내역 타임라인
[화면 하단: 리밸런싱 액션 컨테이너]
현재 보유 중인 5개 ETP 종목의 비중을 보여주는 표(Dataframe) 내장
그 아래에 크게 [목표 비중 리밸런싱 실행 지시서 생성] 버튼 배치. 누르면 텔레그램으로 지시서 발송."
이렇게 **"어떤 컨테이너(상자) 안에 몇 개의 컬럼(칸)을 두고, 무슨 버튼을 넣을지"**만 논리적으로 적어주면 AI가 이를 찰떡같이 파이썬 코드로 번역하여 완벽한 UI를 만들어 냅니다.
