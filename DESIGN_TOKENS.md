# DESIGN_TOKENS.md — 디자인 토큰 단일 소스

이 파일은 프로젝트 전체 색상·타이포·스페이싱·레이아웃의 **단일 소스(Source of Truth)** 다.
AI 에이전트가 UI를 작업할 때 이 파일을 먼저 읽고, 여기 있는 값만 사용한다.
임의 색상 추가·변경 금지. 변경 필요 시 PM 확인 후 이 파일 먼저 수정한다.

연계 문서: `UI_BASELINE.md`, `IMPLEMENTATION_RULES.md`

## 8. AI 에이전트 준수 규칙

1. **이 파일에 없는 hex 값은 사용하지 않는다.**
2. **새 기능 추가 시 이 파일을 먼저 참조한다.**
3. **이 파일을 수정할 때는 PM 확인 필수.**
